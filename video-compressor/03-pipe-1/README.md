# パイプ（１）

通常プロセスは独自のメモリ空間を持っており、他プロセスに直接干渉することは出来ない。

しかし、OSのpipeという機能を使うことで親子プロセス間などで一方通行のデータ送信が可能である。

## os.pipe()

Pythonの`os.pipe()`を呼び出すことにより一時的に制御がカーネルに移る

#### 1.カーネルがパイプ用のメモリバッファを用意する
#### 2.プロセスにパイプ用バッファの書き込み用と読み取り用のファイル記述子(file descriptor)を与える
#### 3.アプリケーション側にfdが返される

プロセスはこのfdを介してメモリバッファに読み書きすることが可能である。

## os.fork()

親子プロセスは同じメモリバッファに対するfdを持っているため、データのやり取りが可能となる。

このpipeという機能によって親子プロセス間でデータの送信が可能となる。
