# Local Chat Messenger(1)

## ソケット

ソケットはパイプと異なり双方向の通信を提供する。

ソケットは特定の`ソケットドメイン`と`ソケットタイプ`に基づいて作成される。

## ソケットドメイン

ソケットドメインはソケットで使用するアドレスの種類を決める。

### AF_UNIX

- 同じコンピュータ内でのプロセス間通信に使用。
- IPアドレスやポート番号ではなく、ファイルパスを介して通信する。
- UNIXドメインソケット、IPCソケット、ローカルソケットなどとも呼ばれる。

### AF_INET

- IPv4を使うネットワークソケット
- 通信先はIPアドレス+ポート番号

### AF_INET6

- IPv6を使うネットワークソケット
- 通信先はIPアドレス+ポート番号

## トランスポート層プロトコル

トランスポート層は、主に、通信の際の両端のコンピュータ間での通信制御を担う。
`TCP`と`UDP`の２種類のプロトコルがある。

## ソケットタイプ

ソケットタイプはソケットがどのような形式で送受信するかを決める。

### SOCK_STREAM

- 信頼性が高く、順序が保たれる
- データは連続したバイト列として扱われるストリーム型の伝送
- AF_INETでは通常TCP

### SOCK_DGRAM

- 基本的に非接続型でデータの到達保証、順序保証、再送制御はなし
- データグラム型(各パケットが独立している)の伝送を提供
- AF_INETでは通常UDP
