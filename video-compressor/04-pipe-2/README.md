# パイプ(2)

## 名前付きパイプ

親子関係のないプロセス間では名前付きパイプを利用することでデータの送受信が可能。

名前付きパイプは無名パイプと異なりファイルパスを介してデータのやり取りを行う。

### os.mkfifo(path, mode=0o600)

`os.mkfifo()`により指定したファイルパスに名前付きパイプという特殊ファイルが作成される。

#### 1.カーネル内部で名前付きパイプ用メモリバッファが確保される
#### 2.アプリケーション側はファイルに対して読み書きを行う
#### 3.内部ではメモリバッファを介してデータのやり取りが行われる

作成された特殊ファイルはos.remove()などで削除しない限り、ファイルシステム上に残るが、
書き込まれたデータはファイル上に残らない。
